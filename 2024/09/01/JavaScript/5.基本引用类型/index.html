5.基本引用类型

kitty2024-09-012025-04-06

引用类型

把数据和功能组织到一起的结构,有时候被成为对象定义,因为它们描述了自己的对象应有的属性和方法

跟类不是一个概念

对象

被认为是某个特定引用类型的实例

构造函数

新对象通过使用 new 操作符后跟一个构造函数来创建,构造函数就是用来创建新对象的函数

ECMAScript 提供了很多像 Date 这样的原 生引用类型，帮助开发者实现常见的任务

RegExp

通过 RegExp 类型支持正则表达式

用\进行转义

实例方法

  • index 字符串匹配模式的起始位置
  • input 要查找的字符串
  • exec() 返回一个匹配信息 没有全局标记调用多少次都是返回第一个匹配的信息
  • test() 接收一个字符串参数 匹配返回 true 否则回 false

构造函数

RegExp 构造函数的所有属性都没有任何 Web 标准出处，因此不要在生产环境中使用它们

原始值包装类型

为了方便操作原始值，ECMAScript 提供了 3 种特殊的引用类型：Boolean、Number 和 String。这些类型具有本章介绍的其他引用类型一样的特点，但也具有与各自原始类型对应的特殊行为

每当用到某个原始值的方法或属性时，后台都会创建一个相应原始包装类型的对象，从而暴露出操作原始值的各种方法

例子

引用类型与原始值包装类型的主要区别

在对象的生命周期

对象在 new 实例化得到的实例会在离开作用域时被销毁

原始值包装对象则只存在于访问它的那行代码执行期间

不能在运行时给原始值添加属性和方法

在原始值包装类型的实例上调用 typeof 会返回”object”

所有原始值包装对象都会转换为布尔值 true

缺点

很容易让开发者疑惑，分不清是原始值还是引用值

Object 构造函数作为一个工厂方法，能根据传入值的类型返回相应原始值包装类型的实例

使用 new 调用原始值包装类型的构造函数，与调用同名的转型函数并不一样

Boolean

要创建一个 Boolean 对象，需要使用 Boolean 构建函数并传入 true 或 false

单例内置对象

概念

任何由 ECMAScript 实现提供，于宿主环境无关，并在 ECMASciprt 程序开始执行时就存在的对象，也就是开发者不用显式的实例化内置对象，因为它们已经实例好了

比如 Object Array String Global Math

Global

详情可以看红宝书 156

在全局作用域中定义的变量和函数都会变成 Global 对象的属性

isNaN()、isFinite()、parseInt()和 parseFloat()，实际上都是 Global 对象的方法

Global 对象属性

Global 对象有很多属性，其中一些前面已经提到过了

像 undefined、NaN 和 Infinity 等特殊 值都是 Global 对象的属性

此外，所有原生引用类型构造函数，比如 Object 和 Function，也都是 Global 对象的属性

window 对象

ECMA-262 没有规定直接访问 Global 对象的方式

但浏览器将 window 对象实现为 Global 对象的代理

因此，所有全局作用域中声明的变量和函数都变成了 window 的属性

当代码开始执行时，全局上下文中会存在两个内置对象：Global 和 Math

所有全局变量和函 数都是 Global 对象的属性

Math 对象包含辅助完成复杂计算的属性和方法

问题

let s = ‘q’ console.log(s.length) 中.length 不是属于引用类型吗？但是为什么我能在原始值这里使用呢

<font style="color:rgb(36, 41, 47);">String</font> 类型在 JavaScript 中是原始值，但它具有一些内置的方法和属性（如 <font style="color:rgb(36, 41, 47);">.length</font>）。当你访问这些属性或方法时，JavaScript 会临时将原始值转换为对象，使你能够使用这些功能。转换完成后，原始值对象会被销毁

所以，虽然 <font style="color:rgb(36, 41, 47);">s</font> 是原始值，但仍然可以使用 <font style="color:rgb(36, 41, 47);">.length</font>

虽然原始值不需要显式的引用类型，但它们的自动包装机制允许你在需要时访问方法和属性

自动包装机制

是 JavaScript 的一个特性，它允许你在处理原始值时，临时将这些值“包装”成对应的对象

例如，当你访问一个字符串的 <font style="color:rgb(36, 41, 47);">.length</font> 属性时，JavaScript 会自动将原始字符串 <font style="color:rgb(36, 41, 47);">s</font> 转换为一个 <font style="color:rgb(36, 41, 47);">String</font> 对象，使你可以访问 <font style="color:rgb(36, 41, 47);">.length</font>，然后再将其转换回原始值

这种机制使得原始值可以像对象一样使用方法和属性
