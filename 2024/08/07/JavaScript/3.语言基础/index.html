3.语言基础

kitty2024-08-072025-04-06

语法

放一些专有名词

标识符

变量，函数，属性或函数参数的名称

严格模式

在整个脚本启用需要在脚本开头加上 use strict

单独指定在一个函数在严格模式下执行，放在函数体开头即可

它是一个预处理指令，任何支持 JS 引擎看到都会切换严格模式

关键字与保留字

有特殊用途 如 break case

未来的保留字 保留给将来做关键字用的 这些都不能做标识符

变量

用于保存任何类型的数据，每个变量只不过是用于保存任意值的命名占位符。如 var const let 声明变量

var，let,const 的区别

  • var 声明的范围是函数作用域，let 是块作用域
  • 当同一作用域内声明两次 let 会抛出 SyntaxError，var 会忽略
  • let 变量不会在作用域中被提升

let age = 19;解析代码时，JS 引擎在声明之前的执行瞬间被称为”暂时性死区”

  • let 在全局作用域中声明的变量不会成为 window 对象的属性，var 会。但是 let 声明也是在全局作用域中发生

var name = ‘a’ console.log(window.name) //‘a’

  • 因为 let 的作用域是块，不会检查前面是否已经使用 let 声明过同名变量，var 会
  • const 是常量 而且声明时必须初始化变量，不能修改会导致运行时错误
  • 赋值为对象的 const 变量不能再被重新赋值为其他引用值，但是对象的键则不受限制 非要变成不可修改 用 Object.freeze() 会静默失败
  • const 声明过程中，JS 运行时编译器可以将其所有实例都替换成实际的值，而不会通过查询表进行变量查找

for 循环中的 let 声明

因为在退出循环时，迭代变量保存的是导致循环退出的值 5

在使用 let 声明迭代变量时，JS 引擎在后台会为每个迭代循环声明一个新的迭代变量。每个 setTimeout 引用是不同的变量实例

声明风格 const 优先，let 次之

const 声明可以让浏览器运行时强制保持变量不变，也可以让静态代码分析工具提前发现不合法的赋值操作

typeof null = object

因为特殊值 null 被认为是一个空对象的引用，表示一个空对象指针

Undefined 与 Null

undefined 是声明变量没初始化时，给变量赋予的值

undefined 是由 null 派生来的

所以 console.log(null == undefined); // true

但是不能显式地将变量值设置为 undefined，但是 null 可以，可进一步将其与 undefined 区分开来

NaN

不是数值的意思，NaN 不等于包括 NaN 在内的任何值

console.log(NaN == NaN);//false

isNaN 该函数会尝试把它转换为数值，不能转换为数值的值会导致这个函数返回 true。

console.log(isNaN(“blue”)); // true，不可以转换为数值

console.log(isNaN(true)); // false，可以转换为数值 1

数据转换

Number()函数转换字符串时相对复杂

parseInt()函数更专注于字符串是否包含数值模式，如果第一个字符是数值字符 +-会继续检测直到碰到非数值字符。如果第一个字符不是数值字符，+-会立刻返 NaN，而 Number()返回 0

parseInt 可以指定转换进制 let num1 = parseInt(‘10’,2) //按二进制解析

parseFloat 跟 parseInt 差不多 但是会忽略开头的 0，只解析十进制，十六进制会返 0，也可以用于字符串转整数

字符串是不可变的

要修改某个变量中的字符串值，必须先销毁原始的字符串

  1. 分配一个足够容纳 10 字符的空间
  2. 填充’java’和’Script’
  3. 销毁原始的两个字符

null 和 undefined 没有 toString()

但是字符串值有这个方法，返回自身的副本

数值调用这个方法时，toString()可以传底数参数

如果不确定一个值是不是 null 或 undefined 可以用 String()

会始终返回表示相应类型值的字符串

  • 如果值有 toString()方法，则调用该方法（不传参数）并返回结果
  • 如果值是 null，返回”null”
  • 如果值是 undefined，返回”undefined”

模板字面量``

字符串插值

字符串插值通过${}中使用一个 JavaScript 表达式实现，是一种特殊的 JS 句法表达式只不过求值后得到的是字符串， 所有插入的值都会使用 toString()强制转型为字符串

原始字符串

用 String.raw()标签函数 但是对实际的操作符是不行的

通过标签函数的第一个参数，即字符串数组的.raw 属性取得每个字符串的原始内容

构造函数

在对象被实例化时自动执行一些必要的操作

派生对象

从一个类继承得到的对象，派生类会继承父类的属性和方法，可以增加自己的属性和方法，也成为子类

symbol 符号类型

符号实例是为唯一，不可变的

使用场景

  • 唯一性->不会和其他属性名冲突
  • 定义全局常量
  • 使用内置的 Symbol 属性
  • 防止属性被意外访问

使用符号作为属性

凡是可以使用字符串或数值作为属性的地方，都可以使用

内置符号

开发者可以直接访问、重写或模拟这些行为。最重要的作用就是重新定义他们，改变原生结构的行为

Symbol.asyncIterator(ES2018)

该方法返回对象默认的 AsyncIterator 由 for-await-of 使用，表示异步迭代器 api 的函数

这个由 Symbol.asyncIterator 函数生成的对象应该通过其 next()方法陆续返回 Promise 实例。可以显式地调用 next()方法返回，也可以隐式地通过异步生成器函数返回

Symbol.hasInstance

判断一个构造器对象是否认可一个对象是它的实例。由 instanceof 操作符使用

Symbol.inConcatSpreadable

一个布尔值，如果是 true，则意味着对象应 该用 Array.prototype.concat() 打平其数组元素

Symbol.inerator

该方法返回对象默认的迭代器。 由 for-of 语句使用, 表示实现迭代器 API 的函数

与 Symbol.asyncInerator 同理

Symbol.match

一个正则表达式方法，该方法用正则表达式去匹配字符串,由 String.prototype.match()方法使用

Symbol.replace

该方法替换一个字符串中匹配的子串。由 String.prototype.replace()方法使用

Symbol.search

该方法返回字符串中 匹配正则表达式的索引。由 String.prototype.search()方法使用

Symbol.species

该函数作为创建派生对象的构造函数，这个属性在内置类型中最常用

用 Symbol.species 定义静态的获取器（getter）方法，可以覆盖新创建实例的原型定义

Symbol.split

该方法在匹配正则表 达式的索引位置拆分字符串。由 String.prototype.split()方法使用

Symbol.toPrimitive

该方法将对象转换为相应的原始 值。由 ToPrimitive 抽象操作使用

很多内置操作都会尝试强制将对象转换为原始值，包括字符串、 数值和未指定的原始类型

Symbol.toStringTag

该字符串用于创建对象的默认字符串描述。由内置方法 Object.prototype.toString()使用

Symbol.unscopables（不推荐）

该对象所有的以及继承的属性， 都会从关联对象的 with 环境绑定中排除

设置这个符号并让其映射对应属性的键值为 true，就可以 阻止该属性出现在 with 环境绑定中

Object 类型

其实是一组数据和功能的集合，通过 new 操作符后跟对象的名称来创建，是所有对象的基类

obj 类型的所有属性和方法在派生的对象上都存在

都有如下属性和方法：

  • constructor：创建当前对象的函数
  • hasOwnProperty(name)：判断当前对象实例(不是原型)上是否存在给定的属性， o.hasOwnProperty(“name”)
  • isPrototypeOf(obj)：用于判断当前对象是否为另一个对象的原型
  • propertyIsEnumerable(name)：判断给定的属性是否可以使用
  • toLocaleString()：返回对象的字符串表示，反映对象所在的本地化执行环境
  • toString()：返回对象的字符串表示
  • valueOf()：返回对象对应的字符串，数值，或布尔值，一般跟 toString()相同

操作符

一元加和减

一元加用到非数值，则会执行与使用 Number()转型函数一样的类型转换：

布尔值 false 和 true 转换为 0 和 1，字符串根据特殊规则进行解析，对象会调用它们的 valueOf()和/或 toString() 方法以得到可以转换的值

按位非

对数值取反并减 1

逻辑或和与具有短路特性

即第一个条件如果不符合条件就不会接下去执行

指数操作符

ECMAScript 7 新增了指数操作符，Math.pow()现在有了自己的操作符**

加法操作符

如果只有一个操作数是字符串，则将另一个操作数转换为字符串然后再拼接在一起

减法操作符

与加法操作符相反，会用 Number()转化为数值

对象则用 valueOf()，没有 valueOf()则用 toString()

关系操作符 > < =

如果有任一操作数是数值，则将另一个操作数转换为数值

任一操作数是对象，则用 valueOf()，没有 valueOf()则用 toString()，再走其他情况

如果有任一操作数是布尔值，则将其转换为数值再执行比较

无法通过 while 实现的逻辑，同样也无法使用 for

for 只是将循环相关的代码封装在了

for-of

是一种严格的迭代语句，遍历可迭代对象的元素

break 和 continue 语句

break 语句用于立即退出循环，强制执行循环后的下一条语句

continue 语句也用于退出循环，但会再次从循环顶部开始执行
